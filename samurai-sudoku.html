How to Play
Basic Rules
  • Five grids: One grid sits in the center, with four more positioned at each diagonal corner, forming an X shape overall.
  • Fill with 1–9: Every cell in each 9×9 grid must contain a digit from 1 to 9.
  • Standard sudoku: Each row, column, and 3×3 box within a single grid must contain every digit exactly once.
Additional Rules
  • Shared corners: Where a corner grid overlaps the center, that 3×3 box belongs to both grids. Digits there must follow the rules of each one.
  • Solve as one: Because the grids are linked through those shared boxes, you have to work across all five at the same time.
