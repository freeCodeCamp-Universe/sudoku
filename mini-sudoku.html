How to Play
Basic Rules
  • The grid: A 4×4 board split into four 2×2 boxes. Fill every cell with a digit from 1 to 4.
  • Rows and columns: Every row and every column must contain each digit exactly once.
  • Boxes: Each 2×2 box must also hold every digit exactly once.
How to Play
  • Given digits: Pre-filled cells are fixed. Only the empty ones can be changed.
  • Entering digits: Click a cell to select it, then press a digit key or tap the numpad.
