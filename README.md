# Sudoku

**Sudoku** is a browser-based collection of multiple sudoku variants, from beginner-friendly 4×4 grids to advanced multi-grid puzzles.

## Development

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
pnpm preview
```

## Test

```bash
pnpm test
```

## Stack

- Vite 7, React 19, TypeScript strict
- CSS modules
- Vitest + @testing-library/react
