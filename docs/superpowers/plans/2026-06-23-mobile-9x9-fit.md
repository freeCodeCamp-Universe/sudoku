# Mobile Fit for 9×9 Boards Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make the classic 9×9 board fit the viewport down to the 320px baseline with no horizontal scroll, while keeping cell touch targets at or above WCAG 2.5.8 AA (24px).

**Architecture:** Replace the ratio-based `window.innerWidth` steps in `useResponsiveCellSize` with an absolute step table that applies **only** to the classic 9×9 grid (`kind === 'grid'`, `size === 9`). Each step is sized so the board width (`9 × cell + 6px` content-box frame) fits the smallest device in its bucket. All other variants (16×16, multigrid) keep their current ratio behavior untouched.

**Tech Stack:** React hook (`useResponsiveCellSize`), Vitest + `@testing-library/react` `renderHook`, jsdom.

## Global Constraints

- Board width formula: `9 × cellSize + 6px` (3px content-box frame each side, no inter-cell gaps). The fit guarantee is `9 × cell + 6 ≤ window.innerWidth`.
- Keep the breakpoint-stepped approach. Do **not** introduce a `ResizeObserver` or fluid container measurement.
- Every classic step must clear WCAG 2.5.8 AA (24px minimum). Floor is 34px.
- The JS thresholds (359 / 413 / 519) are not CSS media queries and do not conflict with the project's `min-width` breakpoint rule.
- Named exports only; no default exports. CSS Modules only if any CSS is touched (none expected). Test file mirrors the unit: `useResponsiveCellSize.test.ts` — never `index.test.ts`.
- Verify before finishing: `pnpm build && pnpm test && pnpm lint` must all pass.

Edit `src/game/useResponsiveCellSize.ts`. Add the two helpers above `useResponsiveCellSize` and branch `compute()`:

```typescript
import { useEffect, useState } from 'react';
import type { Variant } from '@/engine/types';

function getBaseCellSize(variant: Variant): number {
  const kind = variant.layout.kind;
  if (kind === 'grid') {
    const size = (variant.layout as { size: number }).size;
    if (size === 16) return 30;
    return 52;
  }
  if (kind === 'multigrid') {
    const cols = (variant.layout as { canvasCols: number }).canvasCols;
    if (cols === 21) return 30;
    if (cols === 15) return 30;
    if (cols === 12) return 40;
    return Math.floor(400 / cols);
  }
  return 52;
}

function isClassic9x9(variant: Variant): boolean {
  return variant.layout.kind === 'grid' && (variant.layout as { size: number }).size === 9;
}

// Classic 9×9 cell size, stepped so the board (9 × cell + 6px frame) fits the
// viewport down to the 320px baseline. Each step is sized for the smallest
// device in its bucket. See
// docs/superpowers/specs/2026-06-23-mobile-9x9-fit-design.md.
function classicCellSize(width: number): number {
  if (width <= 359) return 34;
  if (width <= 413) return 38;
  if (width <= 519) return 44;
  return 52;
}

export function useResponsiveCellSize(variant: Variant): number {
  const base = getBaseCellSize(variant);
  const classic = isClassic9x9(variant);

  function compute(): number {
    const w = window.innerWidth;
    if (classic) return classicCellSize(w);
    if (w <= 375) return Math.round(base * (38 / 52));
    if (w <= 520) return Math.round(base * (44 / 52));
    return base;
  }

  const [cellSize, setCellSize] = useState(compute);

  useEffect(() => {
    function onResize() {
      setCellSize(compute());
    }
    window.addEventListener('resize', onResize);
    return () => window.removeEventListener('resize', onResize);
    // eslint-disable-next-line react-hooks/exhaustive-deps
  }, [base, classic]);

  return cellSize;
}
```
